## 개요
> ❗***단축키***  
> 💡 <span style='color:green'><b>이클립스에서 자주 쓰이거나 유용해 보이는 단축키 모음.</b></span>  



## 내용
```bash
Ctrl + F6 : 실행 중 탭 목록 

Ctrl + F11 : 실행 

Ctrl + Pgup / Pgdn : 탭 이동

Ctrl + (+) / (-) : 글자 확대/축소

Ctrl + space : 자동완성
- syso : System.out,println();
- For : for 문 자동 완성
- try : try - catch 문 자동 완성 
- fi : finally 자동 완성

​Ctrl + D : 삭제 (영역 지정후 사용시 해당 영역 전체 삭제 / 미지정시 커서가 있는 한줄 삭제)

Alt + 방향키 : 이동 (해당 영역의 소스를 위 아래로 이동)

Ctrl + Alt + 방향키 : 복제(해당 영역의 소스를 위 아래로 복제)

​Ctrl + Shift + F : 자동 들여쓰기

Ctrl + / : java, js 한줄 주석 (//)​

Ctrl + Shift + /  : 여러줄 주석
- java, js  (/* */)
- html (<!-- -->)
- jsp (/* */) 

Ctrl + Shift + \ : 여러줄 주석 해제

Ctrl + Shift + X : 대문자 변환

Ctrl + Shift + Y : 소문자 변환

Ctrl + Shift + L : 모든 단축키 목록 표시

Ctrl + L : 특정 라인으로 이동 

Ctrl + J : Incremental find (한 글자씩 입력할때 마다 일치하는 문자열 찾기) 
한번 더 Ctrl + J 입력후 키보드 위 아래 누르면 이전 / 다음 검색 가능

Alt + Shift + R : 리네임
변수, 클래스를 블럭 지정 하고 실행하면 해당 변수 / 클래스의 이름만 바꿀 수 있다. 
(이름이 같은 다른 변수에는 영향을 주지 않는다)

```
