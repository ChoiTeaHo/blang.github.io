## 개요
> ❗***매번 서버를 재기동 해야 반영되어짐***  
> 💡 <span style='color:green'><b>로컬 톰캣 기준으로 작성했다. 톰캣 설정을 변경하면 된다.</b></span>  



## 내용
### web.xml
```bash
cd /Servers/Tomcat v8.5 Server at localhost-config/web.xml

```

### AS-IS
```xml
    <servlet>
        <servlet-name>jsp</servlet-name>
        <servlet-class>org.apache.jasper.servlet.JspServlet</servlet-class>
        <init-param>
            <param-name>fork</param-name>
            <param-value>false</param-value>
        </init-param>
        <init-param>
            <param-name>xpoweredBy</param-name>
            <param-value>false</param-value>
        </init-param>
        <load-on-startup>3</load-on-startup>
    </servlet>

```

### TO-BE
```xml
    <servlet>
        <servlet-name>jsp</servlet-name>
        <servlet-class>org.apache.jasper.servlet.JspServlet</servlet-class>
        <init-param> <!-- 추가 -->
            <param-name>development</param-name>
            <param-value>true</param-value>
        </init-param>
        <init-param>
            <param-name>fork</param-name>
            <param-value>false</param-value>
        </init-param>
        <init-param>
            <param-name>xpoweredBy</param-name>
            <param-value>false</param-value>
        </init-param>
        <load-on-startup>3</load-on-startup>
    </servlet>

```
