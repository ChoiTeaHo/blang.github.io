## 개요
JS에서 변수 값 초기화 코드를 더 간결하게 만드는것이 목적이다.


## 기본 할당
```js
const example = { a: 123, b: {c: 135, d: 146 }}
const a = example.a;
const b = example.b.d;

```

JSON 데이터를 할당했다.

## 비구조화 할당 
```js
const { a, b: { d } } = example;
console.log(a); // 123  (example 객체의 a 를 GET)
console.log(d); // 146  (example 객체의 b 에서 d를 GET)

```

## 배열 구조분해 할당
```js
arr = [1, 2, 3, 4, 5]
const x = arr[0]        // 1
const y = arr[1]        // 2
const z = arr[4]        // 5
 
// TO-BE
const [x, y, , , z] = arr;

console.log([x, y, , , z]) 
```

할당한 x,y,z 배열의 총 index는 0~4
